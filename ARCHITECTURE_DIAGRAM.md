# Klinflow Architecture Diagram

## Overview
Klinflow is a waste management and recycling platform built as a monorepo using Turborepo. It serves multiple user roles (residents, agents, businesses, admins) with specialized applications for each.

- **Database**: Supabase (PostgreSQL)

- **Storage**: Supabase Storage
- **Edge Functions**: Supabase Edge Functions (Deno)

- **Tool**: Turborepo 2.0.0
- **Package Manager**: npm 11.1.0
- **Workspaces**: apps/*, packages/*

### Resident (Client App)
- Book waste pickups
- Track collection status
- View booking history
- Community features
- Settings management

### Seller (Client App)
- Post material listings
- Manage trades
- Receive and review offers
- AI-powered matching

### Agent (Agent App)
- View available bookings
- Accept/reject jobs
- Track location (H3 geospatial)
- Update booking status
- View earnings

### Business (Business App)
- B2B verification
- Specialized services
- Bulk operations
- License management

### Admin (Admin App)
- User management
- System configuration
- Review moderation
- Analytics dashboard
- Role-based access control

### Hub (Hub App)
- Central coordination
- Cross-app features
- Administrative functions

## Security Model

### Row Level Security (RLS)
- **Profiles**: Public read, owner update
- **Bookings**: User/Agent/Admin access
- **Marketplace**: Active listings public, owner full access
- **Notifications**: Role-based filtering
- **System Config**: Admin-only updates

### Role-Based Access Control
- **user**: Basic resident features
- **agent**: Pickup management
- **business**: B2B features
- **admin**: Full system access

## Summary

Klinflow is a sophisticated waste management platform built with:
- **Monorepo architecture** for code sharing across 6 apps
- **Offline-first PWA** capabilities for reliable operation
- **Real-time synchronization** using Supabase
- **Role-based multi-tenancy** with 4 user types
- **Geospatial features** using H3 indexing for logistics
- **AI-powered matching** for marketplace transactions
- **Comprehensive offline support** with Dexie and Workbox
- **Type-safe development** with TypeScript throughout
- **Modern UI** with TailwindCSS and Framer Motion
